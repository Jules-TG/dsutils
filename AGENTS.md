### This is a library of self-contained reusable functions 

- Python environment is managed with uv
- Codebase contains a range of functions that may or may not be interconnected
- Documentation instrucitons: Google-style docstring with Args: Returns: and Raises: sections -  no type info in the docstring (types are in the signature), summary on the next line as the opening """, and backtick-wrapped inline references.
- Do not edit the content of python functions. Only write and edit documentation.
